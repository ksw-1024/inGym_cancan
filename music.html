その曲、ちょっと、まめ知識

頑張って一人で書きました。良かったら読んでね。



春を告げる / yama

ボカロPのくじらさんが手掛けた楽曲。くじらさん、yamaさんが世に注目されるきっかけになった一曲。

エンジニアを凄腕の人が担当しているので、実は異常に音が良い。


革命 / 黒木渚

黒木渚自身が作詞作曲を手掛けた。

第28回福岡国際クロスカントリー大会のテーマソングでもある。


あったかいんだからぁ / クマムシ

一世を風靡した一曲。「なんだし」もこの人たちが流行らせた。

地味にMVも凝っていて、先入観さえ無ければただの良い曲。


YUMEGIWA LAST BOY / SUPERCAR

いしわたり淳治がギターを務めていたバンド「SUPERCAR」のシングル。

映画「ピンポン」の主題歌に起用されたことでも有名。


星になれたら / Mr.Children

ミスチルがデビューして間もない頃にリリースされた楽曲。一度耳にしたことがある人は多いはず。

リリースから25年が経つが、未だに根強い人気を持つ一曲。


脳漿炸裂ガール / れるりり

VOCALOIDを代表する一曲。人気過ぎて映画になったほど。

まるで早口言葉のように歌詞がならんでおり、活舌が良くないと噛んでしまう。


Dynamite / BTS

知らない人はいないといっても過言ではない、BTSの代表曲。

PV再生数は驚異の14億回。24時間で1億回再生されたButterの7億回再生と比較しても、その人気の根強さが分かる。


ロマンチシズム / Mrs.GREEN APPLE

CMのために書き下ろされた楽曲。ドラムの変化が楽しい。

PVも特徴的なお洒落なものとなっており、話題に。


DAYBREAK FRONTLINE / Orangestar

EDM調の曲。発表当初こそ不評だったものの、「回る空うさぎ」と共に注目を浴びることに。

最高音が非常に高く、常人には歌うことが出来ない。


津軽海峡・冬景色 / 石川さゆり

1977年発表の曲。推薦した人、絶対産まれてない。

石川さゆりがアイドル歌手から演歌歌手へと路線変更したきっかけの曲。


ハッピーウェディング前ソング / ヤバイTシャツ屋さん

ヤバTを代表する一曲。ノリが良く、早口。

「結婚式で2番を歌えない曲」としても有名。


ゲレンデがとけるほど恋したい / 広瀬香美

「アルペン」CM、並びに映画「ゲレンデがとけるほど恋したい」のダブルタイアップ。

音がぴょんぴょん跳ねる。難しい。


『バッハの旋律を夜に聴いたせいです。』 / サカナクション

本校野球部、伝統の歌。らしい。

サカナクションの楽曲の中でも飛び抜けた「カオス」が光る。


ロマンスの神様 / 広瀬香美

誰もが知る、広瀬香美の代表作。

最近はTikTokの音源として用いられる事も多く、再び注目を浴びている。


天城越え / 石川さゆり

カラオケに行くと、誰かは歌う曲。時代を超えて愛されている。

ビブラートがつけやすく、加点を狙いやすい。


2億4千万の瞳-エキゾチックジャパン- / 郷ひろみ

郷ひろみ、50作目のシングル。THE FIRST TAKEに出演したことで再び脚光を浴びる。

紅白歌合戦でも毎年恒例の歌となっている。


High Hopes / Panic! At The Disco

TikTokで日本でも注目を浴びた、知る人ぞ知る名曲。

Gabriela Beeのカバーが音楽の楽しさを体現した動画となっている、


青春"サブリミナル" / =LOVE

THE "アイドル"の曲。

昭和アイドルを彷彿とさせるキャッチーな曲。


Gee / 少女時代

韓国で大ヒットを記録した曲。街中で耳にした人も少なくないはず。

初めて恋に落ちた女性の気持ちを表現したファストテンポの曲。


世界をかえさせておくれよ / サンボマスター

「できっこないを やらなくちゃ」に並ぶサンボマスターの代表曲。

いろはすのCMソングにも起用された。


五等分の気持ち / 中野家の五つ子

「五等分の花嫁」のOPソングとして制作された。

歌っている声優が豪華。尊い。


紅蓮華 / LiSA

「鬼滅の刃」のOP。紅白歌合戦にもこの曲で出演。

「炎」と合わせて、LiSAの代表曲となっている。


Pretender / Official髭男dism

言わずとしれた曲。何なら説明不要。

マスタリングを世界的エンジニアのTed Jensenが担当。彼はONE OK ROCKの「Eye of the Storm」も担当したすごい人。


海賊戦隊ゴーカイジャー / 松原剛志

2005年生まれの私達だと、観ているか観ていないか微妙なライン。

35周年のレンジャーであり、ゴーカイブルーは山田裕貴。


Town / アイドルマスター

YouTubeのBGMで流れていることが多い。

YouTubeを普段観ていれば必ず耳にしていることだろう。


ニコニコ動画流星群 / しも

ニコニコで人気のある楽曲をメドレー形式でつなぎ合わせた楽曲。

有志で集ったネットユーザーオーケストラで演奏されたことでも有名に。


君と夏フェス / SHISHAMO

「SHISHAMOといえば」の一曲。

軽音楽部がバンドで演奏するイメージがある。気がする。


まるまるもりもり / 薫と友樹、たまにムック。

芦田愛菜と鈴木福のコンビが歌っている。小学校のときに踊った人も多い。

今改めて聴いてみると、歌詞がちょっと沁みる。


おばけもどき / わーるど feat.ぺんぺん

(諸事情により、推薦文を一字一句変更せず掲示いたします。)

わーるどの2ヶ月振りの新曲。

ボーカルにはぷらそにか大阪のぺんぺん氏を抜擢。わーるどがボーカルに人を登用するのは史上初のケース。

ぺんぺん氏は持ち前のかわいらしい声でファンから絶大な人気を博している。

YOASOBIボーカルの幾多りら氏も過去に所属していた、超人気アコースティックセッションユニット、ぷらそにかとのコラボ曲ということで世間からの注目度も高く、再生数も投稿から10日で700回と驚異的な数字を残している。

今話題沸騰中の「おばけもどき」。一度聞いてみてはいかがだろうか。


夏色 / ゆず

ゆずがメジャーデビューすることになった代表曲。今なお多くの人に愛されている。

テレビではよく「夏色」を路上ライブしていたときの様子が放送されている。


Woman〜Wの悲劇より〜 / 薬師丸ひろ子

映画「Wの悲劇」の主題歌。

映画の主演を努めた薬師丸ひろ子が歌唱を担当している。


丸の内サディスティック / 東京事変

おしゃれ楽曲の創始者といっても過言ではない。

この楽曲で用いられた和声の進行は、最近の楽曲で多用されている。


PPAP / ピコ太郎

ジャスティンビーバーがハマった曲らしい。

彼の日本に対するイメージがおかしな方向に変わっていないことを切に願う。


サイレントマジョリティー / 欅坂46

アイドルに詳しくなくても、この曲は知っているという人が多い。

平手友梨奈はアイドルの中でも飛び抜けて輝いている。（個人の見解）


会いたかった / AKB48

「ヘビーローテーション」と共にAKBの名を全国に知らしめた一曲。

ダンスも特徴的で、流行っていた時には誰もがサビの振り付けを真似していた。


Happiness / 嵐

嵐の楽曲の中でも特に前向きな一曲。

リリースは2008年。今もなお色褪せないメロディとなっている。


Love so sweet / 嵐

花より男子2の主題歌としても知られる。

サビがあまりにも有名。それほどまでにキャッチーなフレーズが用いられている。


Shape of you / Ed Sheeran

洋楽にあまり馴染みのない人でも、この曲は知っている。そのくらい流行った。

カラオケで歌う前に、必ず和訳歌詞を読むことを強く推奨する。


RAGE OF DUST / SPYAIR

銀魂の主題歌担当としても知られるSPYAIRが手掛けた一曲。

ガンダムのオープニングとしても知られる。


お願いマッスル

人気過ぎてYouTubeにて1億回再生を突破した。多分アニメ観た人よりこの曲聴いた人のほうが多い。

なかやまきんに君のカバーもおすすめ。


かくれんぼ / Whiteberry

「夏祭り」でよく知られるWhiteberryの曲。

「ピカチュウのドキドキかくれんぼ」のオープニングテーマ。


あいつこそがテニスの王子様

元はミュージカル・テニスの王子様の代表的な一曲。

あまりに空耳が聴こえるので、ニコニコ動画を中心に空耳コメントで盛り上がった。


さくらんぼ / 大塚愛

言わずと知れた一曲。自然と、手を叩きたくなる衝動に駆られる。

同じメロディを何度も繰り返すことで相手の記憶に刷り込む、昔ながらのマーケティング手法がうかがえる。


XY&Z / サトシ & 松本梨香

ジガルデやサトシゲッコウガが登場した時期のテーマソング。

サトシゲッコウガの人気は高く、今なお多くのファンアートが描かれている。


キラキラ / 佐香智久

作曲者の佐香智久は、少年Tという歌い手としても人気を集める。

ポケモンとのタイアップは数多く、「ゲッタバンバン」が有名。


Blessing / halyosy

伝説の歌い手と呼ばれるhalyosyが、ニコニコラボと銘打って発表した楽曲の1つ。

「ニコニコラボ」文化は大きく人気を集め、一大コンテンツに。


レッツゴー！陰陽師 / でんちゅう

ニコニコ動画内にて現存する最古の動画。ニコニコユーザーなら必ず通る道。

本来はPS2向けゲームのオマケ映像だが、その奇怪さから注目を集めた。


明日への道 〜Going my way!!〜 / 遠藤正明

遊戯王のオープニングテーマとして有名。

アニメとマッチした楽曲として評価が高い一方、空耳でネタにされることも。


Get Wild / TM NETWORK

サビだけならみんな歌える。非常に有名な曲。

リリースは1987年、この頃はまだレコードが主流だった。


NIGHT OF FIRE / NIKO

パラパラブームの最大の火付け役。と言ってもみんな分からないと思うが…。

キムタクが自身の番組で取り上げたことで一躍有名に。


ウィーアー！/ きただにひろし

言わずと知れたワンピースのテーマソング。アニメソングでは伝説の曲と称される。

冒頭にサビを持ってくる、現代も多く用いる構成となっており、未だに根強い人気がある。


閃光 / [Alexandros]

「機動戦士ガンダム 閃光のハサウェイ」の主題歌。

ガンダム人気は再び上がっており、ガンプラの生産が間に合っていない。


小さな恋のうた / MONGOL800

「お〜い〇〇、バンドやろうぜ！」となると大抵候補にあがる曲。

楽器類は比較的簡単に演奏ができるものの、ボーカルの技量を高く求められるため、曲選びの時は気をつけよう。


パプリカ / Foorin

米津玄師プロデュース、NHK2020応援ソングプロジェクトの曲。

実はオリンピックの曲ではない。オリンピックテーマは嵐の「カイト」。これも米津玄師がプロデュースしている。


赤いスイートピー / 松田聖子

松田聖子の楽曲の中でも特に人気が高い。作曲はユーミンこと松任谷由実。

作詞担当の松本隆が「ライバルに曲を書かないか？」と松任谷本人に打診し、この曲が生まれた。


完全感覚Dreamer / ONE OK ROCK

ONE OK ROCKの名を世に知らしめた一曲。

PV内でアンパンマンドラムを叩く、替え歌「完全在宅Dreamer」を本人たちが発表し、話題になった。


前前前世 / RADWIMPS

「君の名は。」のテーマソングとして一躍有名に。

最近では18祭の「正解」が話題になった。はず。


キセキ / GReeeeN

皆が知っている、GReeeeNの代表的な曲。カラオケで歌えば年代問わず盛り上がる。

最近の歯医者さんは凄いんだなぁ。


Everyday、カチューシャ / AKB48

前田敦子センターの楽曲。2011年リリース。

作編曲は、東京2020オリンピック聖火リレー公式BGMを制作した事でも知られる、井上ヨシマサが担当した。


女々しくて / ゴールデンボンバー

おそらく日本で一番有名なエアーバンド。紅白に出れないことでも有名。

カラオケで歌う人が多く、今でも沢山の印税が入ってくるらしい。


粉雪 / レミオロメン

「1リットルの涙」の挿入歌。2005年リリース。

歌うとちょっとしんみりする。


3月9日 / レミオロメン

粉雪と並ぶレミオロメンの代表作。

実は卒業ソングではなく、結婚ソングだという逸話がある。


令和 / ゴールデンボンバー

令和が発表されて、ものの数時間で公開された曲。

数々の民法の音楽番組に出演したものの、この年も紅白への出場はならず。


メルト / supercell

halyosyさん(Blessingの作者)が歌ったことでniconico動画のランキングを埋め尽くした伝説の歌。

ryoさんは後に「君の知らない物語」も制作する。


語彙力が足りない / なきそ

TikTokで注目を浴びた「げのげ feat.ロス」の生みの親、なきそのデビュー作。

Sony MusicのPuzzle Projectのメンバーの1人でもある。これ大事。


赤い電車 / くるり

ロックバンド・くるりの曲。

赤い電車とは、京急の電車のこと。品川駅の接近メロディとして用いられている。


/ / // / / / x0o0x_

題名がない。そのため検索がしにくく、おすすめに流れてこない限り聴くことができなかった。

便宜上、「ワカメ」と呼ばれることがある。


蕾 / コブクロ

YouTubeでは8000万回以上再生されている超人気曲。

この曲に限らず、コブクロの楽曲はハモリが美しく、デュエットの練習曲として選ばれることも多い。


アスノヨゾラ哨戒班 / Orangestar

ウォルピスカーターの歌ってみたが、niconico歌ってみたランキングの1位であることでも有名。

Orangestarの作品の中では比較的難易度が低く、カラオケでも人気がある。


U.S.A. / DA PUMP

流行りに流行った。乗りやすい曲調、真似しやすいダンスも相まって社会現象に。

Joe YellowのUSAのカバーであるが、あまり知られていない。


チェリー / スピッツ

(素晴らしい推薦文をいただいたため、そのまま掲載します。)

「ポケットモンスター 赤・緑」が発売され、「Yahoo! JAPAN」がサービスを開始し、羽生善治が将棋タイトル7冠を達成した1996年に発表されたスピッツ屈指の名曲。

某動画サイトでは再生回数が9000万回を超え、今もなお、沢山の人々に愛されている。


恋 / 星野源

「逃げるは恥だが役に立つ」のEDテーマ。

男女の恋愛にとらわれず、多くの形の「恋」に当てはまる歌詞が高く評価され、話題に。


時よ / 星野源

メロディで魅せにきた一曲。

MVも特徴的で、ファンから根強い人気がある。


SUN / 星野源

ノーカットで撮られたPV、秀逸な編曲、歌いやすいメロディで星野源の名が世広まった。

恋と合わせて、彼の代表的な一曲となる。


Week End / 星野源

特徴的なサビで、アルバム限定曲にも関わらず話題に。

大阪城ホールでのライブ映像が公開されている。


夢の外へ / 星野源

2012年リリース。資生堂CMソングとして世の記憶に残る。

変拍子や遊び心のある編曲で注目される。Goose houseのカバーも話題に。


ultra soul / B'z

サビを知らない人はいない。ただ、B'z史上最高の売上という訳ではない。

世界水泳にこの曲が用いられ、一気に知名度を上げた。


A・RA・SHI / 嵐

国民的アイドルグループ、嵐のファーストシングル。

2019年にYouTubeにて公開されたPVでは、あのHIKAKINもコメントしている。


Monster / 嵐

「怪物くん」の主題歌。同作では大野智が主演を務めた。

作曲者は、NHK Eテレ「ノージーのひらめき工房」に出演するタノチーミー。


いつも何度でも / 木村弓

「千と千尋の神隠し」の主題歌。ライアーとの弾き語りの姿が印象的。

ゆらぎのあるリズムと音程が耳に心地よく、今なお多くの人に愛される。


Pop Virus / 星野源

星野源 5st Albumの表題曲。既存の価値観にとらわれない、非常に攻めた曲だが、ヒット作に。

名作PS4ゲーム「DEATH STRANDING」に楽曲が使用され、配達中に耳にすることが出来る。


Tie Me Down / Elley Duhe & Gryffin

リリース自体は2018年だが、今年に入りTikTokでBGMとして用いられる事が増え、現在はUniversal Musicのトップ再生回数を誇っている。

ストレートな流れの音楽にもやもやとした歌詞が多くの人の心を掴んだ。


金木犀 feat.Ado / くじら

Adoがまだ「うっせぇわ」をリリースする前に発表された曲。このころはまだAdoのTwitterのフォロワーは4000人ちょっとだった。

「春を告げる」に並び、くじらの代表的な曲となっている。


これからのこと / ケプラ

TikTokで冒頭が有名になり、一気に注目が集まる。

味のあるバンドサウンドが魅力的。


立ち上がリーヨ / T-Pistonz

2008年にリリースされた曲。イナズマイレブンOPとしても有名。

無性に元気が出る曲で、かつてイナズマイレブンを観ていたファンを中心に人気がある。


イロトリドリ / ゆず

ゆずの人気曲。いきものがかりとコラボしたことで再注目される。

当時は珍しい、3DCGを多用したMVとなっていて、言葉通り「イロトリドリ」な作品となっている。


プラスチックアドベンチャー / MusMus

いわゆる「フリーBGM」。多くの動画投稿者がこのBGMを使用したことから、ほとんどのネットユーザーが知る一曲となる。

あまりに使われ過ぎたせいで逆に古臭さを感じてしまう、ある意味かわいそうな曲である。


エゴロック / すりぃ

一分の曲。あまりに人気が出たので3分になった。

中毒性が高く、ループから抜け出せないファンが大量に発生した。

負けないで / ZARD

言わずと知れた応援ソング。運動会での定番。

今なお多くの人に愛される。


行くぜっ!怪盗少女 / ももいろクローバーZ

ももクロのメジャーデビューシングル。教科書にも載っている。

作詞作曲をヒャダインこと前山田健一が担当。


東京サマーセッション / HoneyWorks

「告白実行委員会」シリーズの一曲。良く見ると歌っている声優が実力派ぞろいだったりもする。

2021年に東京スプリングセッションが発表され、四季全ての「セッション」シリーズが揃った。


烈車戦隊トッキュウジャー / 伊勢大貴

トッキュウジャーのOPソング。ちなみに私、観ていたので歌えます。

ひたすらに真っ直ぐな曲調で、イマジネーションを力に戦っていたトッキュウジャーにぴったりな曲。


Be The One / PANDORA feat.Beverly

小室哲哉プロデュース。小室サウンドとBeverlyの鋭い歌声がマッチし、テンションの上がる曲に仕上がっている。

仮面ライダービルドは1話だけでも観てほしい。めっちゃ面白い。


シンデレラガール / King Prince

キンプリデビュー作。「花のち晴れ」の主題歌。


マリーゴールド / あいみょん

あいみょんの代表曲。ギターの演奏の難易度も高くないので、よく弾き語りで歌われる。

あいみょんはスマホのフリック入力が異常に早い。


Make you happy / NiziU

NiziUのデビュー曲。縄跳びダンスでも話題になった。

配信シングルのみで紅白歌合戦に出場した珍しいグループ。
